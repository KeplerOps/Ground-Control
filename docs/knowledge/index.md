# Ground Control Knowledge Base Index

Flat catalog of every page in this repository's knowledge base, grouped by
category. Before creating a new page, an agent MUST grep this index for
existing coverage of the topic and, if a page already exists, update it
rather than creating a duplicate. See `SCHEMA.md` for the full conventions.

Format of each entry is:

```
- [page-title](relative/path.md) — one-line summary. Tags: tag1, tag2
```

## Topics

_No pages yet. Topic pages live under `topics/` and cover cross-cutting
subjects that span multiple files or entities._

## Entities

_No pages yet. Entity pages live under `entities/` and document concrete
code entities (classes, modules, endpoints, migrations, config files)._

## Gotchas

_No pages yet. Gotcha pages live under `gotchas/` and document specific
pitfalls and their resolutions._
